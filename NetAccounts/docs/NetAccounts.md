# NetAccounts Module
## Description
Manages local security groups and local user accounts on local and remote Windows computers on the Windows platform.

## NetAccounts Cmdlets
### [Add-NetLocalGroupMember](Add-NetLocalGroupMember.md)
Adds members to a local security group.

### [Disable-NetLocalUser](Disable-NetLocalUser.md)
Disables local user accounts.

### [Enable-NetLocalUser](Enable-NetLocalUser.md)
Enables local user accounts.

### [Get-NetLocalAdminGroup](Get-NetLocalAdminGroup.md)
Gets the local Administrators security group.

### [Get-NetLocalAdminUser](Get-NetLocalAdminUser.md)
Gets the local Administrator user account.

### [Get-NetLocalGroup](Get-NetLocalGroup.md)
Gets local security groups.

### [Get-NetLocalGroupMember](Get-NetLocalGroupMember.md)
Gets members of local security groups.

### [Get-NetLocalUser](Get-NetLocalUser.md)
Gets local user accounts.

### [Get-WellknownNetPrincipal](Get-WellknownNetPrincipal.md)
Gets well-known principals.

### [New-NetLocalGroup](New-NetLocalGroup.md)
Creates a local security group.

### [New-NetLocalUser](New-NetLocalUser.md)
Creates a local user account.

### [Remove-NetLocalGroup](Remove-NetLocalGroup.md)
Removes local security groups.

### [Remove-NetLocalGroupMember](Remove-NetLocalGroupMember.md)
Removes members from a local security group.

### [Remove-NetLocalUser](Remove-NetLocalUser.md)
Removes local user accounts.

### [Rename-NetLocalGroup](Rename-NetLocalGroup.md)
Renames a local security group.

### [Rename-NetLocalUser](Rename-NetLocalUser.md)
Renames a local user account.

### [Set-NetLocalGroup](Set-NetLocalGroup.md)
Changes local security groups.

### [Set-NetLocalUser](Set-NetLocalUser.md)
Changes local user accounts.
